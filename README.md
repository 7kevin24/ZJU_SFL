# ZJU SFL

一个基于 [Streamlit](https://streamlit.io/) 的轻量级应用。

>  **Note**: 本项目约 90% 的代码由 Google Gemini 生成。

##  环境配置

请确保安装了 Conda，然后并在终端中执行以下命令来创建环境并安装依赖：

```bash
# 1. 创建并激活 Python 3.13 环境
conda create -n SFL python=3.13
conda activate SFL

# 2. 安装依赖
pip install -r requirements.txt
````

## 运行

环境配置完成后，执行以下命令启动服务器：

```bash
python -m streamlit run app.py
```
